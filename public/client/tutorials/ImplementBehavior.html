Implement Function

Given the function description, you have 10 minutes to implement this function fully/partially. You can write/edit the code in the function editor, create/edit/remove tests cases and run the tests suite on your code.

Function Implementation Complete

Once you have implemented the function completely, you can click the checkbox in the test editor panel as shown below and then submit this task.

Function Implementation Incomplete

If you have partially implemented the function, please describe the unfinished sections as pseudo code to notify the next programmer.

Comment Code

Please write the code with appropriate comments as this will help the following programmers understand your code quickly.

Calling a function

To see the list of all functions in the system, type ALT+SPACE (Mac) or CTRL+SPACE (Windows/Linux)

Creating a new function

If you need a supporting function to realize the implementation, you can create a new function and submit your task. This function will then be assigned to programmers to implement. To create a new function, type ALT SPACE(Mac) or CTRL SPACE(Windows/Linux) and select “Add a new function”, where you can write a comment specifying the function name, description, parameters, parameter types and return type.
