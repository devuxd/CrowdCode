Implement Function

Given the function description, you have 15 minutes to implement this function fully/partially. You can write/edit the code in the function editor, create/edit/remove tests cases and run the tests suite on your code.

TIP: Everywhere in website you can open the tutorials by clicking on the question mark icons.

Function Implementation Complete

Once you have implemented the function completely, you can click the checkbox in the test editor panel as shown below and then submit this task.

coding tasks (implement one behavior tasks)

Step 1: find only one behavior in the function description(in the comments above the function) that is not yet implemented, for example: "it should check arguments to be valid"

Step 2: write a test for the behavior you have chosen implement the behavior in the function editor, just implement one bahvior, it does not need to implement more than one bahvior. For example: only implement "it thow TypeError if argmunents are null or empty."

Step 3: implement the behavior in the function editor, just implement one bahvior, it does not need to implement more than one bahvior. For example: only implement "it thow TypeError if argmunents are null or empty."

Calling a function

To see the list of all functions in the system, type OPTION+SPACE (Mac) or CTRL+SPACE (Windows/Linux)

Creating a new function

If you need a supporting function to realize the implementation, you can create a new function and submit your task. This function will then be assigned to programmers to implement. To create a new function, type ALT SPACE(Mac) or CTRL SPACE(Windows/Linux) and select “Add a new function”, where you can write a comment specifying the function name, description, parameters, parameter types and return type.
