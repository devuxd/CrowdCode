Calling a function

To see the list of all functions in the system, type ALT+SPACE (Mac) or CTRL+SPACE (Windows/Linux)

Creating a new function

Sometimes, behaviors are complicated and you want to create a new function which is responsible for that behavior. If you need a supporting function to realize the implementation, you can create a new function and submit your task. This function will then be assigned to OTHER programmers to implement, you should NOT implement the new function, you should just write a description in the below of main function and call the new function in the body of your main function and finally submit it. To create a new function, use shortcut keys OPTION + SPACE(Mac) or CTRL + SPACE(Windows/Linux) and select “Add a new function”.
