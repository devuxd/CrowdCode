Test editor

In the test editor you can use ChaiJs website via Link.

example: 

var res = calculate('+',[1,2]);
             expect(res).to.deep.equal(3);
var badFn = function () { throw new TypeError('Illegal salmon!'); };
                expect(badFn).to.throw(TypeError);

While editing the test code, you can open the auto-complete by pressing ALT+SPACE (Mac) or CTRL+SPACE (Win/Linux).
