Tutorial 2: you will faced with two types of tasks to accomplish. type A: coding only one behavior; type B: reviewing a code and tests which are implemented by other programmers. Read the below screenshots

Type A: coding tasks (implement one behavior tasks), follow 4 bellow steps in screenshots for each task of coding

Step 1: find only one behavior in the function description(in the comments above the function) that is not yet implemented, for example: "it should check arguments to be valid"

Step 2: write a test for the behavior you have chosen

Step 3: implement the behavior in the function editor, just implement one bahvior, it does not need to implement more than one bahvior. For example: only implement "it thow TypeError if argmunents are null or empty."

Step 4: once you’re finished, click “Submit” button.

Type B: review tasks (reviewing code of the other programmer), follow 4 bellow steps in screenshots for each task

Step 1: review implementation of the behavior which is written by other programmer

Step 2: review tests of this behavior

Step 3: rate the contribution and give a feedback to the development

Step 4: once you’re finished, click “Submit” button

If you are ready click “Get Started!”
