2: You will faced with two types of tasks to accomplish. Read the below workflow

Type A: Implement behaviour tasks

Step 1: Find a behavior in function description that is not yet implemented
Step 2: write a test for this behavior
Step 3: Implement the behaviour in the function editor
Step 4: Congrats! Submit the task

Type B: Review tasks

Step 1: review implementation of the behavior
Step 2: review tests of this behavior
Step 3: rate the contribution and make a comment
Step 4: Congrats! Submit the task
