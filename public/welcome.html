Programming, re-envisioned

Microtasks

Crowd Microservices organizes programming into microtasks, small units you can pick and do like writing a few lines of code and brain- storming tests.

Start it, or finish it!

Write some pseudocode, leave it for the crowd. The crowd will write the functions as well as test it for you.

Automatic task generation

Crowd Microservices tracks what needs to be done, generating and managing microtasks.

Code, Tested

All code is unit tested.
When the tests pass, it’s ready to go.

Your work, your credit

Points

Microtasks are worth points; the harder the work, the more points it’s worth.

Reviews

Microtasks you submit are reviewed by the crowd. If accepted, you get the points; if rejected, you get nothing. You get points to review others work as well.
