Use the pseudocode to sketch something with the syntax '//# pseudocode'. 


	


	

		Use the pseudocalls to request other functions
		adding a line of description, the header and an empty body for the 
		requested function at the bottom of the code editor.
		

	

	

		Note: all function calls are pass by value (i.e., if you pass an
		object to a function, and the function changes the object, you will not
		see the changes).
