Want to sketch something? Write a line of pseudocode with '//#'

Want to call a function? Add its header with a line of description after the body of the current function and call it without worring about its implementation (you can't implement it here).
The crowd will look for a matching function or eventually create it if doesn't exist yet.

Note that all function calls are pass by value (i.e., if you pass an object to a function, and the function changes the object, you will not see the changes).
