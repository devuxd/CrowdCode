Write test cases tutorial

In this microtask you are asked to write some test cases starting from a function signature.
what is a test case?

“A test case is a set of conditions under which a tester will determine whether an application, software system or one of its features is working as it was originally established for it to do” [Wikipedia]

For CrowdCode, you don’t need to formally specify a test case but you are instead encouraged to write a simple description in natural language.
