Reuse search

In Reuse Search, your goal is to identify a function best matching the specified function call, or determine that no such function yet exists.

To search for functions, you can type text in the textbox. So, to find a sum function, you might search for “sum”. Of course, a function might be described differently, so you might need to try other queries.

After finding a matching function, select the function to choose it. Or if no such functions exists, select “No function found”.
