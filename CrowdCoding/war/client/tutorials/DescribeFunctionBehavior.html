Describe a function behavior

given the function description, your goal is to add one or more test cases to programmatically describe its behavior

for example given the function EXAMPLEFUNCTION, some behaviors can be

  • it should return null if the parameter is invalid
  • another test 1
  • another test 2
  • another test 3

to add a new behavior click on 'add a new test' and the add form will be displayed

you can choose either to implement the test as 'simple test' or as 'advanced test'

simple tests are I/O tests where you have to specify the output of the function when a specified set of inputs is provided

advaced tests allows you to check if a function result contains a determined property, .......
Advanced tests in CrowdCode are implemented following the ChaiJs expect syntax.
